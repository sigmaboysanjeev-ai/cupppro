# CUPP-PRO (Markov Edition)

CUPP-PRO is an **ethical password wordlist generator** inspired by CUPP.
It generates **weak to strong, human-realistic passwords** based on
OSINT-style inputs and orders them using a **Markov probability model**.

üìå This tool **only generates wordlists**.
It does **NOT** perform cracking or attacks.

---

## Features

- CUPP-style interactive inputs
- Weak ‚Üí medium ‚Üí strong password coverage
- Name + name + date modeling
- Partial birthdate logic
- Relationship-based patterns
- SaaS & seasonal trends
- Markov ordering (best guesses first)
- Deduplicated output
- No external dependencies
